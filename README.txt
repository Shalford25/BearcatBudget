Bearcat Budget is a booking software for small businesses, built with C# and uses MySQL for the databases. WinUI was used for the design of the application and it was done as a project for CIS 499 at Lander University.

Required Plugins/Add-Ons:
* Live preview

Folders for the website include:
* data/www (where all the html files go)
* data/assets (where all the assets go)
